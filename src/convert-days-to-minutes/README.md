# Convert Days to Minutes

This tool converts the number of days to minutes.

## Example

```text
Days: 1
Minutes: 1440
```

## How to convert days to minutes

To convert days to minutes, multiply the number of days by 1440.

```text
1 day = 1 * 1440 = 1440 minutes
```
