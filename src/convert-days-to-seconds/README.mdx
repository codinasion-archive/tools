# Convert Days to Seconds

This tool converts the number of days to seconds.

## Example

```text
Days: 1
Seconds: 86400
```

## How to convert days to seconds

To convert days to seconds, multiply the number of days by 86400.

```text
1 day = 1 * 86400 = 86400 seconds
```
